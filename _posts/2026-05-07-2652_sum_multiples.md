🔍 **LeetCode Problem: 2652. Sum Multiples**

আজ আমি একটা interesting math-based problem solve করেছি যেখানে
`1` থেকে `n` পর্যন্ত যেসব number `3`, `5`, অথবা `7` দিয়ে divisible, সেগুলোর sum বের করতে হবে।

---

# 🧠 Main Intuition

এই problem-এ দুইটা গুরুত্বপূর্ণ concept use হয়েছে:

1️⃣ Arithmetic Sum Formula
2️⃣ Inclusion-Exclusion Principle (LCM based)

---

# 🧩 প্রথমে pattern বুঝি

ধরো:

```text
n = 10
```

3 দিয়ে divisible numbers:

```text
3, 6, 9
```

এগুলোকে অন্যভাবে লিখলে:



🔥 এখানেই আসল trick।

# 🧠 Arithmetic Sum Formula

আমরা জানি:

```text
1 + 2 + 3 + ... + n
```

এর formula হলো:

```cpp
n * (n + 1) / 2
```

---

# 🔍 Example

যদি:

```text
count = 3
```

তাহলে:

```text

=
3 × (3 + 1) / 2
=
3 × 4 / 2
=
12 / 2
=
6
```

# ⚠️ কিন্তু এখানে confusion হতে পারে



তাই শেষে formula-টাকে `num` দিয়ে multiply করতে হয়।

মানে:

# ⚠️ Overlapping Problem

এখানে একটা সমস্যা আছে।

```text

```

👉 এটা:

তাই এটা দুইবার add হয়ে যাবে ❌


# 🧠 Final Formula (Inclusion-Exclusion)

```cpp
sum3 + sum5 + sum7
- sum15 - sum21 - sum35
+ sum105
```

---

# ⚡ Full Code (C++)

```cpp
class Solution {
public:
    long long sumDivisibleBy(int num, int n) {

        long long count = n / num;

        return num * (count * (count + 1) / 2);
    }

    int sumOfMultiples(int n) {

        long long sum3 = sumDivisibleBy(3, n);
        long long sum5 = sumDivisibleBy(5, n);
        long long sum7 = sumDivisibleBy(7, n);

        long long sum15 = sumDivisibleBy(15, n);
        long long sum21 = sumDivisibleBy(21, n);
        long long sum35 = sumDivisibleBy(35, n);

        long long sum105 = sumDivisibleBy(105, n);

        return sum3 + sum5 + sum7
             - sum15 - sum21 - sum35
             + sum105;
    }
};
```

# 🧠 Important Understanding

👉 Formula:

```cpp
count * (count + 1) / 2
```

শুধু দেয়:

```text
1 + 2 + 3 + ...
```

কিন্তু actual multiples বানানোর জন্য:

```cpp
num *
```

করতে হয়।

---

# ⚠️ Another Important Thing

অনেকে ভাবে:

```text
2 * 3 / 2
```

মানে:

```text
2 * (3 / 2)
```

কিন্তু সেটা ভুল ❌

কারণ:

* `*` and `/` same precedence
* evaluate হয় left → right

অর্থাৎ:

```text
(2 * 3) / 2
=
6 / 2
=
3
```
